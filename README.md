#Excel Price Automation
#Overview
Python script that reads an Excel file, applies a 10% discount to prices, writes updated values, and creates a bar chart.

#Tech
Python
openpyxl

#Features
Reads Excel data
Applies discount (10%)
Writes corrected prices
Generates bar chart

#Run
Bash
pip install openpyxl
python main.py

#Notes
Price must be in column 4
Output saved in same file
Sheet name: Sheet1
